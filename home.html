Project Outline: SSSP(single-source shortest path problem) is well-known graph algorithms, which has been proved can be implemented with time complexity O(nlogn + m) on a graph with n nodes and m edges using the well-known Dijkstra algorithm. As for the Parallel version of SSSP algorithms, the Δ-stepping algorithm and the Radius-stepping algorithm are two existing solutions for this problem. However, both of these 2 algorithms have their problem. The Δ-stepping algorithm can not find the worst-case bound on general graphs, whereas, the Radius-stepping doesn't have any efficient implementation available. So this paper described a new stepping algorithm framework, and a new data structure called lazy-batched priority queue (LaB-PQ ) to implement a better version of parallel SSSP algorithm with non-trivial worst-case bounds.
